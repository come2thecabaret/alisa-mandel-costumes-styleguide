Introduction

The purpose of this document is to outline a style guide for the Alisa Mandel Costume Design website. Similar to how you provide fabric samples to clients, this page is designed to outline the colors, typography and animation styles for your Costume Design site.

NOTE: The document itself does not layout how the site will look, but rather how components will be themed across the site. Doing this not only allows you as the customer to start to see how the site is coming together, but also allows the developer to pre-emptively start putting together assets.

Colors
Base Palette

Due to this project focusing around colorful outfits and costumes, a neutral color palette was deemed best in order to highlight the costumes.

White (#ffffff)
This will be used for the main background of the site.
Beige (#e5ded4)
This will be used to highlight gallery items (see page layout below).
Black (#333333)
This will be used as the main font color for paragraphs on the white or beige background.
